# 삼각함수로 2차원 선형변환 분석하기 (2)

---

### 질문

- 2차원 선형변환에서 변환 전 벡터의 각도와 변환 후 벡터의 각도는 어떤 관계가 있나?

---

시작하기 전에 각도 표현을 정리해보자. 이제부터 각도들은 x축으로부터의 각도가 아니라 주축으로부터의 각도로 생각해보자. 즉 선형 변환 후에 타원의 장축이 되는 입력 벡터가 있는데, 이제 입력 좌표계에서의 각도 $\theta$는 이 벡터로부터의 각도로 생각하자. 그리고 출력 좌표계에서의 각도 $\theta^\prime$은 타원의 장축으로부터의 각도로 생각하자.

이전 페이지에서 봤을 때, 변환 후 벡터의 길이 제곱은 $C + R \cos(2\theta)$로 표현할 수 있었다. 변환 후 타원의 장축의 길이를 a, 단축의 길이를 b라고 할 때, $C + R = a^2$, $C - R = b^2$ 이다. 왜냐면 이 수식은 다르게 보면 중심이 C이고 반지름이 R인 원 위를 돌아가는 벡터로 생각할 수 있기 때문임. 즉 길이 제곱은 이 원 위의 벡터의 cos 값인데, 이 벡터는 x축으로 C만큼 가고, 거기서 180도마다 돌아가는 벡터이다. cos값이 가장 작을 때가 길이 제곱이 가장 작을 때이고, 가장 클 때가 길이 제곱이 가장 클 때이고, 가장 작을 때는 단축이고, 가장 클 때는 장축이 됨. 즉 $\theta$가 0일 때 벡터는 장축으로 변환되고, 90도일 때 단축으로 변환된다.

이번엔 변환 후 좌표계에서 생각해보자. 타원의 방정식은 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 이다. 그리고 타원의 중심에서 타원 위 임의의 점까지의 거리를 r이라고 하자. 점의 위치에 따라서 r은 변하는 값이고, 해당 점의 좌표는 $(r\cos(\theta^\prime), r\sin(\theta^\prime))$ 으로 표현할 수 있다. 이걸 타원 방정식에 넣어보면,

$$
\begin{aligned}
r^2 (\frac{\cos(\theta^\prime)^2}{a^2} + \frac{\sin(\theta^\prime)^2}{b^2}) = 1 \\
r^2 = \frac{a^2 b^2}{(b^2 \cos^2(\theta^\prime) + a^2 \sin^2(\theta^\prime))}
\end{aligned}
$$

근데 $\cos^2(\theta^\prime) = \frac{1 + \cos(2\theta^\prime)}{2}$ 이니까, $b^2 \cos^2(\theta^\prime) = \frac{b^2 (1 + \cos(2\theta^\prime))}{2}$ 이다. 그리고 $\sin(\theta^\prime) = \cos(\frac{\pi}{2} - \theta^\prime) = \cos(\theta^\prime - \frac{\pi}{2})$ 니까 $a^2 \sin^2(\theta^\prime) = a^2 \cos^2(\theta^\prime - \frac{\pi}{2}) = \frac{a^2(1 + \cos(2\theta^\prime - \pi))}{2}$ 이다. 즉 $2\theta^\prime$ 방향으로 가고 그거에 정확히 반대방향으로 가는 벡터를 생각해볼 수 있는 것. 즉 이 두 값을 더한 거는 결국에 정리하면 $\frac{(a^2 + b^2) + (b^2 - a^2) \cos(2\theta^\prime)}{2}$가 되는 거. 그리고 이건 C와 R로 표현하면 $C - R \cos(2\theta^\prime)$. 그러면 기존 식은 다음과 같이 정리됨.

$$
r^2 = \frac{a^2 b^2}{C - R \cos(2\theta^\prime)}
$$

그러면 변환된 벡터의 길이제곱은 변환 전 좌표계의 각도 $\theta$에 대한 함수로 표현하면 $C + R \cos(2\theta)$ 이고, 변환 후 좌표계의 각도 $\theta^\prime$에 대한 함수으로 표현하면 $\frac{a^2 b^2}{C - R \cos(2\theta^\prime)}$이다. 이 두 함수들을 두 가지 관점에서 봐볼 건데, 첫 번째는 각도가 서로 같은 상황에서 함숫값을 비교해보기, 두 번째는 함숫값이 서로 같은 상황에서 각도를 비교해보기.

첫 번째로 각도가 서로 같은 상황에서 함숫값을 비교해보기. 만약에 $\theta = \theta^\prime = k$ 라고 하자. 이 각도에서 다음과 같은 부등식을 가정해보자.

$$
\frac{a^2 b^2}{C - R \cos(2k)} <= C + R \cos(2k)
$$

전개해보면,

$$
a^2 b^2 <= C^2 - R^2 \cos^2(2k)
$$

인데, $a^2b^2 = C^2 - R^2$ 이니까 위의 부등식은 다음과 같이 정리됨.

$$
cos^2(2k) <= 1
$$

정리하면 입력 좌표계에서 각도 k인 입력 벡터가 변환되어서 출력 좌표계로 갔다. 그 벡터의 길이는 $C + R \cos(2k)$ 이다. 근데 이 변환된 벡터가 출력 좌표계에서 어떤 각도인지는 몰라. 별개로 출력 좌표계에서 각도 k인 벡터가 있었다. 이 벡터의 길이는 $\frac{a^2b^2}{C - R \cos(2k)}$ 였다. 부등식을 보니까 $\frac{a^2b^2}{C - R \cos(2k)} < = C + R \cos(2k)$가 성립한다. 따라서 입력 좌표계에서 각도 k인 벡터가 변환된 길이는, 이미 출력 좌표계에 있던 각도 k 벡터의 길이보다 항상 큰 거야, 단 주축에서는 같음. 

두 번째로 함숫값이 서로 같은 상황에서 각도를 비교해보기. 이건 그래프로 봐보자.

x축은 각도이고 y축은 길이제곱이다. 0도는 장축이 되니 가장 크고, 90도는 단축이 되어서 가장 작다. 파란색은 $C + R \cos(2\theta)$ 이고, 빨간색은 $\frac{a^2b^2}{C - R \cos(2\theta^\prime)}$ 이다. 두 각도를 동일한 x축에 겹쳐서 그렸다. 

이전 부등식에서 본 관점은 y축에 평행한 선을 긋고 함숫값이 어떻게 변하는지 봤던 거였다. 이번에 함숫값이 같은 상황에서 각도 변화를 본다는 건 x축에 평행한 선을 긋고 본다는 거야. 

0도에서 90도로 가는 걸 봐보자. 같은 함숫값인 상황에서는 빨간색의 각도가 더 작아지는 걸 알 수 있다. 반면에 90도에서 180도 방향에서는 같은 함숫값에서 빨간색의 각도가 더 커진다. 이걸 상상해보면 장축에서 단축으로 갈 때는 변환 후 각도가 계속 작다가, 단축에서 장축으로 갈 때는 더 큰 각도로 변환된다는거.

이걸 고유벡터 관점에서도 생각해볼 수 있는데, 다음과 같은 그림을 위의 그래프와 같이 보자. 주의할 건 밑의 그림에서 나오는 각도는 이 페이지에서 지금까지 얘기했던 각도랑 다르게 주축 기준 각도가 아니라, x축 기준 각도로 통일한거임.

![입력좌표계와 출력좌표계의 각도차이와 정확하게 일치.png](https://minseok127.github.io/docs/%EC%84%A0%ED%98%95%EB%8C%80%EC%88%98/입력좌표계와_출력좌표계의_각도차이와_정확하게_일치.png)

 $v_1$은 $u_1$이 되고, $v_2$는 $u_2$가 된다. $u_1, u_2$는 타원의 주축들이다. $e_1, e_2$는 선형 변환의 고유벡터들이다. 이걸 위의 그래프와 같이 생각해보면, 이전 그래프 상에서의 0도는 주축으로부터의 각도를 의미하기에 $v_1, u_1$을 의미하고 90도는 $v_2, u_2$를 의미한다.

$v_2$가 $u_2$로 변환되는 지점에서 시작해보자. 여기서 각도가 커진다면, 출력 벡터가 $u_1$과 이루는 각도는 입력 벡터가 $v_1$과 이루는 각도보다 점점 커진다. 즉 위의 그림에서 출력 벡터는 $u_2$에서 점점 이동하고 입력 벡터는 $v_2$에서 점점 이동하는데, 원래는 $v_2$가 더 각도가 컸으니까 위의 그림 상으로는 출력 벡터 각도가 더 작아 보이다가, 점점 입력 벡터 각도를 따라잡게 되는 거지. 그러다가 둘이 만나게 되는 지점이 생기면, 이 각도가 고유벡터가 되는 지점이 된다.

이걸 이전 그래프 관점에서 보려면, 먼저 위의 그림에서 파란 직교축을 빨간 직교축으로 회전시켜서 겹친 후에 생각해보면 된다. 겹쳐진 관점에서 봤을 때, 90도 각도에서 180도 각도로 파란 벡터와 빨간 벡터가 출발하는데, 빨간 벡터가 더 빠르게 돌아간다. 그러면 빨간 벡터와 파란 벡터의 각도 차이가 이전 그래프 상에서 초록색 막대 길이가 된다. 그러다가 두 벡터의 차이가 $v_1$과 $u_1$의 차이만큼 벌어진다면? 즉 초록 막대 길이가 $v_1$과 $u_1$의 각도 차이가 된다면? 그 지점에서 다시 직교축을 겹치기 전으로 되돌린다면, 파란 직교축이 $v_1$과 $u_1$의 각도 차이만큼 다시 되돌려지게 되고, 그러면 파란 벡터 (입력 벡터)와 빨간 벡터 (출력 벡터)가 벌어졌던 것이 상쇄되어서 같은 방향이 된다.
