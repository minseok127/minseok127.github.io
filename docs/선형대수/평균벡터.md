# 평균을 벡터로 생각해보기

---

### 질문

- 평균을 벡터로 어떻게 생각할 수 있나?
- 편차를 벡터로 어떻게 생각할 수 있나?

---

n개의 데이터 값들이 있다. 이걸로 n차원 벡터를 생각해보자. 벡터의 성분들이 데이터 값들이다. 그러면 이 데이터들의 평균을 벡터 관점에서 볼 수 있을까? 평균이라는 거는 결국 데이터 전체를 대표하는 하나의 값을 의미한다. 다르게 말하면 모든 데이터들을 하나의 값으로 치환해서 생각하겠다는 것이다. 즉 평균을 벡터로 생각한다는 것은 n차원 데이터 벡터의 모든 성분을 동일한 값으로 바꾸겠다는 것이다.

n차원 벡터인데 모든 성분값이 1인 벡터를 1벡터라고 부르자. 평균 벡터는 이거를 평균값으로 스케일한 것이다. 즉 데이터 벡터를 평균 벡터로 만든다는 것은 결국 1벡터 방향으로 치환한다는 것이다. 그러면 평균 벡터는 원래 벡터와 어떤 관계를 가질까? 원래 벡터 x를 1벡터로 정사영 시킨 벡터가 k라는 성분값들을 가진다고 치자. 이 벡터의 길이는 $k\sqrt{n}$ 이다. 1벡터의 길이는 $\sqrt{n}$이다. 정사영 벡터 길이와 1벡터 길이를 곱한 값은 원래 벡터 x와 1벡터를 내적한 값이므로, 이를 정리하면 다음과 같다.

$$
k\sqrt{n} \times \sqrt{n} = 1^T\vec{x} = x_1 + x_2 + ... \ + x_n
$$

따라서 k는 평균 값이 되고, 다르게 말하면 원래 벡터 x를 1벡터 방향으로 정사영 시킨 벡터가 성분값들이 평균값으로 이뤄진 평균 벡터라는 것이다. 그리고 원래 벡터와 이 정사영 벡터를 뺀 벡터가 편차들을 성분으로 가지는 벡터가 되니까 빗변이 원래 벡터 x이고 밑변이 평균벡터인 직각 삼각형에서 높이 방향이 편차 벡터가 되는 것이다. 좀 더 정확히는 평균벡터에서 원래 벡터로 가는게 편차 벡터이다.

이 직각삼각형을 생각해보면 분산도 생각할 수 있다. 편차 벡터를 e라고 표현하고 평균을 $\bar{x}$라고 표현할 때, 피타고라스 정리에 의해 원래 데이터 벡터 x의 길이 제곱은 평균 벡터 길이 제곱 + 편차 벡터 길이 제곱이므로, 수식을 세워보면 다음과 같다.

$$
 \lVert \vec{x} \rVert^2 = n \times \bar{x}^2 +  \lVert \vec{e} \rVert^2
$$

여기서 양변에 n을 나누면 결국 수식은 다음과 같이 변한다.

$$
E[X^2]=(E[X])^2+Var(X)
$$
